#Emmabooks

##Development

Clone this repository, then use nvm to use the correct version of node:

    nvm use

After this, run:

    npm install
    bundle install

##Start!

It's easy:

```
bundle exec middleman
```

##Build

```
bundle exec middleman build
```

##Deploy
Just push. Emmabooks is hosted on Netlify: every time you push, site is builded.
